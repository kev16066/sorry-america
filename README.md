# Sorry, America — World Cup 2026 watchlist

A small, comment-verified collection of YouTube videos of international World Cup 2026
visitors discovering everyday U.S. culture, plus a section on visitors reconciling with
media narratives about American safety and people.

Static single-page site (`index.html`). No build step, no backend, no tracking added by us;
videos play via YouTube's standard embed and belong to their creators.

## Deploy (GitHub Pages)
Push this folder to a repo, then enable Pages on the default branch (root). Done.
